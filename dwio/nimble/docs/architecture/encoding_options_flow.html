Encoding tree (zeroCopy=true path only)
NullableEncoding(pool, data, sbf, const Options& options)
  base copies options → options_
  nimble::EncodingFactory(options).create(pool, ...)  // local, always base class
    → DictionaryEncoding(..., options)
      nimble::EncodingFactory(options).create(pool, ...)
        → TrivialEncoding(..., options)
          Compression::uncompress(..., options.decompressCounter(), bufferPool)
▶
Decompression time tracking only works on the zeroCopy=true path. The legacy factory drops options (including decodingStats) in its 4-arg override. Once zeroCopy becomes the prod default, tracking works everywhere.
▶
All 12 production encodings use this pattern: ALP, Dictionary, Delta, Nullable, MainlyConstant, FreqPartition, ForEncoding, RLE, BBP, SubIntSplit, Trivial (string), DeltaEncoding.
